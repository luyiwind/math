题目与打印
这个页面主要围绕“生成-练习-核对-打印”这一流程设计，操作起来很直观。我们来看看几个亮点：

智能出题规则：题目严格遵循“100以内”和“不进位、不退位”的要求，系统会自动检查个位与十位的加减是否满足条件。同时，题目会随机混合两数运算和三数运算（如12+23+15=__），避免练习过于单调。

灵活的填空位置：除了常规的“求结果”（如23+15=__），题目还会随机将空格放在算式前面（如__+15=38或23-__=8）。这样能锻炼孩子逆向思考的能力，而不仅仅是机械计算。

打印友好设计：点击“打印题目”按钮或使用浏览器打印功能时，工具栏会自动隐藏，题目会以适合A4纸的格式清晰排版，方便您直接打印出来给孩子做纸面练习。

答案核对模式：点击“显示答案”按钮，答案会直接出现在题目空白处和页面底部的答案区。核对时一目了然，再次点击即可隐藏，方便反复练习。
